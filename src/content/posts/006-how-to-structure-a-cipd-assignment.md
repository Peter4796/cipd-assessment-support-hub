A clear structure does two things: it keeps your thinking organised, and it makes your work easy for an assessor to mark against the criteria. A well-structured average answer often outperforms a brilliant but chaotic one. Here's a structure that works across CIPD levels.

## 1. A focused introduction

Set the scope, signpost what the response will cover, and where relevant introduce the organisational context. Keep it tight: three or four sentences that orient the reader.

## 2. Criteria-led main sections

Organise the body around the tasks and assessment criteria, not around a story. Use clear headings that map to the brief. This guarantees you address every criterion and makes it obvious to the assessor where each is met.

> Use the brief's own language in your headings. If a task says 'analyse approaches to reward', a heading like 'Analysis of reward approaches' signals exactly where that criterion is answered.

## 3. Apply theory to practice

Within each section, move from theory to application. Introduce the model or concept, then show how it plays out in a real workplace example. Application is where higher marks live, especially at Levels 5 and 7.

## 4. A conclusion that adds value

Summarise the key points and, where the brief asks for it, make clear, justified recommendations. Don't introduce new material here. Draw the threads together.

## 5. A complete reference list

Finish with a full, consistent Harvard reference list. Every in-text citation should appear here, formatted correctly.

## Plan before you write

1. List every assessment criterion.
2. Draft headings that map to each task.
3. Allocate a word budget to each section.
4. Note the models, evidence and examples you'll use.
5. Only then start writing.
